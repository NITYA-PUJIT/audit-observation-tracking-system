# Audit Observation Tracking Frontend

React + Tailwind frontend for the Audit Observation Tracking System.

## Run

```powershell
npm install
npm run dev
```

The app runs on `http://127.0.0.1:5176` and proxies `/api` to the backend at `http://127.0.0.1:4310`.

## Build

```powershell
npm run build
```
